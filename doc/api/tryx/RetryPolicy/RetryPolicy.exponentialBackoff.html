Creates a retry policy with exponential backoff.

This is a convenience constructor that sets up exponential backoff with sensible defaults. The delay between attempts will increase exponentially: initialDelay, initialDelay * multiplier, initialDelay * multiplier^2, etc.

Example:

final policy = RetryPolicy.exponentialBackoff(
  maxAttempts: 5,
  initialDelay: Duration(milliseconds: 100),
  backoffMultiplier: 2.0,
  maxDelay: Duration(seconds: 10),
);
