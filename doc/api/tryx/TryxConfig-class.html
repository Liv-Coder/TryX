TryxConfig

TryxConfig class

Global configuration for the Tryx library.

This class provides a centralized way to configure default behaviors for error handling, logging, retry policies, and other aspects of the Tryx library. The configuration is applied globally and affects all operations unless explicitly overridden.

Example:

// Configure global settings
TryxConfig.configure(
  defaultRetryPolicy: RetryPolicies.network,
  enableGlobalLogging: true,
  logLevel: LogLevel.warning,
  globalErrorMapper: (error) => 'Global: $error',
);

// Use safe operations with global config applied
final result = safe(() => riskyOperation());
