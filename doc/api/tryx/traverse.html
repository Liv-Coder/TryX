Traverses a list of values, applying a function that returns a Result.

This is similar to List.map but for functions that return Result. If all applications succeed, returns a Success containing a list of all results. If any application fails, returns the first Failure.

This is useful for applying a potentially failing operation to each element in a list, such as parsing or validation.

Example:

final strings = ['1', '2', '3', 'invalid'];
final results = traverse(strings, (s) => safe(() => int.parse(s)));

results.when(
  success: (numbers) => print('All parsed: $numbers'),
  failure: (error) => print('Parse failed: $error'),
);
