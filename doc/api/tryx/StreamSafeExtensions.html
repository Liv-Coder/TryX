StreamSafeExtensions

StreamSafeExtensions<T> extension

Extensions for working with Stream instances and converting them to safe streams.

These extensions provide functionality to convert regular streams into streams of Result instances, enabling safe stream processing without traditional try-catch blocks in stream transformations.
