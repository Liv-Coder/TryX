Converts a nullable value to a Result.

If value is not null, returns a Success containing the value. If value is null, calls errorProvider and returns a Failure with the provided error.

This is useful for converting nullable values from APIs or databases into the Result type for consistent error handling.

Example:

String? maybeString = getValue();
final result = fromNullable(
  maybeString,
  () => 'Value was null',
);

result.when(
  success: (value) => print('Got: $value'),
  failure: (error) => print('Error: $error'),
);
